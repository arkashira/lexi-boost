# Lexi Boost

A VS Code extension that offers inline code completion for Python, Go, and TypeScript.

## Features

* Registers a completion provider for Python, Go, and TypeScript
* Completion suggestions appear after 200 ms of typing
* Can be installed via the VS Code marketplace

## Usage

1. Install the extension via the VS Code marketplace
2. Open a Python, Go, or TypeScript file in VS Code
3. Start typing and completion suggestions will appear after 200 ms
